# Driving School Management System

A Bash-based CLI application designed to simulate the management of a driving school. This project focuses on efficient data handling and process automation using shell scripting.

## Key Features
- **User Authentication:** Secure Login/Logout system for administrators.
- **Candidate Management:** Full CRUD operations (Create, Read, Update, Delete) for student records.
- **Reporting:** Automated generation of student status reports.
- **Data Persistence:** Information is stored and managed using local text-based databases.

## Tech Stack
- **Scripting:** Bash
